# Piano MIDI Visualizer

一个轻量的钢琴 MIDI 可视化器 + 文件回放 + 练习模式（错音检测）+ AB 循环 + 录制导出 + 视奏训练。单文件，零依赖。

## 功能

### 实时输入
- **MIDI 输入**：通过 Web MIDI API 自动识别已连接的 MIDI 键盘（Chrome / Edge 推荐）。
- **电脑键盘 fallback**：`A S D F G H J K L ;` 白键，`W E T Y U O P` 黑键，`Z / X` 切换八度，`N` 切音名，`P` 切练习模式，空格 播放/暂停。
- **力度可视化**：音符透明度随 MIDI velocity 变化。

### MIDI 文件回放
- **拖拽 / 选择 / Demo 一键加载** `.mid` 文件，自动解析 SMF format 0 / 1。
- **瀑布流示范**：白色亮音显示正在响的音符，淡蓝显示预览音符。
- **Transport**：播放/暂停、进度条、Tempo 25%-200%、循环、卸载。

### 4 个内置 Demo 曲目
- **Twinkle Twinkle Little Star**（旋律 + I/IV/V 和弦伴奏）
- **Ode to Joy**（贝多芬第九交响曲主题）
- **Für Elise 开头**（贝多芬 Bagatelle WoO 59）
- **Happy Birthday**

### 多声道分色（按 track 上色）
设置面板里切换 3 种模式：
- **C4 split**：默认，以中央 C 为界，右手金色 / 左手青色
- **by Track**：按 MIDI track 染色（6 色调色板：金/青/紫/玫瑰/天蓝/琥珀），看多 track 文件时一眼区分声部
- **single**：单色

### 练习模式（错音检测）
- **打开**：点头部「练」按钮，或按 `P`。
- **实时对照**：示范音符会按状态变色——
  - 🟢 绿色：你弹对了（在 ±300ms 容差内匹配上）
  - ⚪ 灰色：你错过了
  - ⚪ 白色（发光）：示范正在响
  - 🔵 淡蓝：未来音符
- **错音红闪**：错音 → 对应键红色闪烁 0.6 秒
- **时间偏差显示**：`+12ms` 或 `-45ms`，告诉你弹早还是弹晚
- **统计面板**：命中数 / 错过数 / 错音数 / 准确率 / 平均时间偏差

### AB 循环
transport 栏的 `A` `B` 按钮标记循环区间，✕ 清除。

### 录制 + 导出
- 🔴 **录按钮**：录制中红色脉冲
- **▶ 回放**：把刚才录的内容加载为示范
- **↓ 保存**：导出为标准 `.mid` 文件（SMF format 0），可在 DAW / Synthesia / Piano Marvel 里打开

### 视奏训练（新）
点头部 **视奏** 按钮一键生成符合调性的随机练习，或点 ⚙ 配置：
- **调号**：C / G / D / A / E / F / B♭ / E♭
- **调号位**：oct 3 / 4 / 5
- **难度**：
  - 入门：C 大调风格，全级进，单八度内，无跳进
  - 中级：可有三度音程、八分音符
  - 高级：可有五度以上跳进、宽音域（接近两个八度）
- **小节数**：2 / 4 / 8
- **BPM**：40-160
- **自动播放**：生成后立即开始

#### 生成器音乐规则
- 起止音是主音（解决乐句）
- 强拍（4/4 的 1/3 拍）偏好和弦音（I/IV/V）
- 弱拍偏好经过音 / 邻音
- 大部分级进运动，偶尔三度、罕见五度（按难度）
- 和声进行从 I / IV / V / vi / iii 中随机选

### 示范音色
内置 Web Audio 加法合成（基波 + 二次/三次谐波），ADSR 包络 + 低通滤波。

## 已验证（端到端测试通过，28 项）

**回归**：Twinkle 78 音符 / 旋律开头 7 音正确

**多声道分色**：split / track / none 三模式都对，isNow 永远白色

**视奏生成质量**：
- 4 bars / beginner：16 音符 / 7.96 秒
- 起止音都是主音 (60)
- **最大 leap ≤ 4 semitones**（beginner 上限）
- advanced 8 bars：39 音符 / range 23 semitones（接近两个八度）
- G 大调起止音都是 67 (G4)
- **同 seed 完全确定，不同 seed 5 个唯一**
- **beginner 0 个非调内音**（全 diatonic）
- MIDI 导出回环一致
